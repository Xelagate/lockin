# LockIn — P2P Ескроу на Solana

> Довірчий ескроу-сервіс для P2P угод крипто ↔ фіат, побудований на Solana.  
> Colosseum Hackathon 2026.

## Огляд

LockIn дозволяє двом сторонам обмінювати криптовалюту на фіат (банківський переказ, готівка тощо) без взаємної довіри. Смарт-контракт утримує токени в ескроу до підтвердження отримання фіату продавцем — усуваючи ризик контрагента.

## Як це працює

```
Продавець створює угоду → поповнює ескроу → Покупець приймає → платить фіат офлайн
→ Продавець підтверджує отримання → токени передаються Покупцю
```

Якщо щось іде не так — будь-яка зі сторін може відкрити спір, який вирішує адмін-ключ.

## Життєвий цикл угоди

```
Created → Funded → Accepted → Completed
                             → Cancelled
                  → Disputed → Resolved
```

| Статус      | Опис                                                      |
|-------------|-----------------------------------------------------------|
| `Created`   | Угода створена, очікується депозит від продавця           |
| `Funded`    | Продавець задепозитив токени в ескроу PDA                 |
| `Accepted`  | Покупець приєднався, фіатний платіж в процесі             |
| `Completed` | Продавець підтвердив отримання фіату, токени передано     |
| `Cancelled` | Угода скасована до `Accepted`, токени повернено продавцю  |
| `Disputed`  | Відкрито спір, очікується рішення адміна                  |
| `Resolved`  | Адмін вирішив спір                                        |

## Інструкції смарт-контракту

| Інструкція         | Хто викликає | Опис                                          |
|--------------------|--------------|-----------------------------------------------|
| `create_deal`      | Продавець    | Створює PDA угоди з сумою та токен-мінтом     |
| `fund_deal`        | Продавець    | Депозитить SPL-токени в ескроу                |
| `accept_deal`      | Покупець     | Приєднується як покупець, бере зобов'язання   |
| `confirm_receipt`  | Продавець    | Підтверджує отримання фіату → передає токени  |
| `cancel_deal`      | Продавець    | Скасовує угоду, повертає токени продавцю      |
| `dispute_resolve`  | Адмін        | Вирішує спір на користь однієї зі сторін      |

## Технічний стек

- **Смарт-контракт**: Rust + [Anchor 0.32](https://www.anchor-lang.com/)
- **Блокчейн**: Solana (devnet / mainnet-beta)
- **Токени**: SPL Token — USDC / USDT
- **Фронтенд** *(в розробці)*: Next.js 14 + Tailwind CSS
- **Гаманець**: Phantom через `@solana/wallet-adapter`
